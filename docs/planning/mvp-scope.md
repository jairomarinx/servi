# MVP Scope - servi.cam

## IN (v1)
- User registration and auth
- Person profile with optional national_id
- Provider and client role activation
- Skills assignment
- Service creation by providers (local focus)
- Service search by category and city
- Service request creation by clients
- Feed ranked by skill match
- Proposal system
- Direct booking
- Review and rating
- Basic trust score

## OUT (not v1)
- Payment intermediation
- National ID verification API
- Mobile app
- Remote services as primary focus
- Team providers
- Recurring services
- Push or SMS notifications (email only)
